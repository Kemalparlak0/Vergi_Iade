# Vergi İade Projesi

Bu projede vergi iade süreci otomatikleştirildi.
Video linki: [Video Linki]

## Kullanılan Teknolojiler
- React
- HTML / CSS / JavaScript

## Nasıl Çalıştırılır
1. `npm install` ile gerekli paketleri yükleyin.
2. `npm start` ile projeyi başlatın.
3. Tarayıcıda http://localhost:3000 adresinden görüntüleyin.
